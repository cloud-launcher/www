cloud-launcher is a set of free and open-source tools for designing, launching, and managing distributed computing systems on a global scale.
cloud-launcher aims to integrate with every provider on the planet to not only free you from reliance on a single provider, but to allow you to run across any and all providers to create real high-availability systems that can survive in all but the most catastrophic scenarios.
